# AtheistWorldToken Frontend

A React app for interacting with the AWT smart contract.

## Setup
1. npm install
2. npm start

## Features
- Wallet connection with MetaMask/TrustWallet
- Buy, stake, unstake tokens
- Referrals and rewards
